# TP1 – Open Science

## Présentation
Ce dépôt a été créé dans le cadre du TP1 du module Open Science.  
Il a pour objectif de découvrir les principes du développement collaboratif
et l’utilisation de Git et GitHub.

## Objectifs du TP
- Comprendre les bases de Git
- Utiliser GitHub pour gérer un dépôt
- Découvrir les principes de l’Open Science
- Travailler de manière collaborative

## Outils utilisés
- GitHub
- Git
- Navigateur web (travail réalisé depuis un téléphone mobile)

## Contenu du dépôt
- README.md : description du projet

## Remarque
Le travail a été réalisé via GitHub en ligne (mobile).
Les commandes Git ont été étudiées de manière théorique.
